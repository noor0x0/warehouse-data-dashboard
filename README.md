# Warehouse Data Dashboard

## Overview
This project is a mini data warehouse and dashboard system built using collected warehouse-related data.  
It includes data preparation, ETL processing using Python, SQL scripts, and a dashboard interface for displaying warehouse insights and summaries.

## Project Components
- Data collection and preparation
- ETL process using Python
- SQL database scripts and queries
- Web-based dashboard interface
- Data visualization and reporting

## Technologies Used
- Python
- PHP
- SQL
- MySQL
- HTML
- CSS
- JavaScript
- Data Cleaning
- ETL

## Features

Warehouse data management
Data cleaning and transformation
SQL-based storage and querying

Tables, summaries, and visual reports
ETL Process

## The ETL process includes:

Extracting collected warehouse data
Cleaning and preparing the dataset
Transforming the data into a structured format
Loading or preparing the data for dashboard usage
Database

The SQL/ folder contains database scripts, table creation queries, and related SQL operations used in the project.

## Dashboard

The Dashboard/ folder contains the web dashboard files used to display warehouse data and summaries.

## Data

The Data/ folder contains the collected warehouse-related dataset used in this project.
